# Subroutine 

1. In Scala, a user can create function with or without = (equal) operator.

2. If the user uses it, the function will return the desired value.
   ```$xslt
     def myfun1normal( a:Int, b: String):String ={
         println(b+ a)
         return b+a+"!.";
       }
   ```
